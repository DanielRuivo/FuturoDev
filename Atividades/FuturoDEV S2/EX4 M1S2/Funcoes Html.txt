<!DOCTYPE html>Serve para definir o tipo de documento HTML.
<html>É a variavel que envolve todo o código HTML.
<head>Serve para definir as informações sobre o documento, como titulo, links tato para CSS quanto para JavaScript, e outras funções.
<title>Define o título do documento, que aparece na aba do navegador.
<body>Serve para mostrar o conteúdo do documento, ou seja, o que será exibido na tela.
<h1> a <h6>Serve para definir os títulos da pagina ou como textos, do grande para o pequeno, onde o <h1> é o mais importante e <h6> o menos importante.
<p>Serve para exibir um parágrafo.
<br>Serve normalmente para criar uma linha nova, mas também pode ser usado para criar um espaço entre os textos.
<hr>Serve para criar uma linha horizontal, normalmente usada para separar os textos.
<strong>Serve para deixar o texto em negrito
<em>Serve para muadar o tipo da fonte para itálico.

<a>Serve para criar um link, onde o atributo href define o endereço do link.
<img>Servve para adicionar uma imagem ao documento, onde o atributo src define o endereço da imagem.
<ul>Serve para fazer uma lista não ordenada que nao tem uma ordem definida, ou seja, não tem números.
<ol>Serve para fazer uma lista ordenada, ou seja, tem números.
<li>Serve para definir um item dentro de uma lista, tanto a ordenada quanto a não ordenada.
<table>Serve para criar uma tabela onde as outras tags são usadas para definir o conteúdo da tabela.
<tr>Define uma linha na tabela. Cada linha pode conter várias células como <td> ou <th>.
<td>Serve apra definir uma linha dentro da tabela.Serve para conter o conteúdo da célula
<th>Serve para definir o cabeçalho da tabela, ou seja, o título da coluna.
Geralmente usada como primeira linha ou colunapara descrever o conteudo da tabela (normalmente exibido em negrito e centralizado).  

<thead>Serve para definir o cabeçalho da tabela. Contém as linhas e células de cabeçalho, tanto em colunas quano em linhas.
<tbody>Serve para definir o corpo da tabela. Contém as linhas e células de dados principais.
<tfoot>Serve para definir o rodapé da tabela. Contém as linhas e células de resumo ou totais.
<caption>Serve para definir o título da tabela. Normalmente exibido acima da tabela, deve ser colocado logo após a tag <table>.
